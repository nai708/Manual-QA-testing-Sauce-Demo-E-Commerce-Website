# Manual-QA-testing-Sauce-Demo-E-Commerce-Website
Developed and executed 30+ manual test cases for login, cart, checkout, sorting, and form validation functionality. 

## Objectives
The objective of this project was to practice manual software testing by identifying defects in an e-commerce testing environment.

Testing focused on:
- Login and authentication
- Shopping cart functionality
- Checkout validation
- Item sorting
- Navigation and UI behavior

| bug-reports.docx | Documented defects discovered during testing |
